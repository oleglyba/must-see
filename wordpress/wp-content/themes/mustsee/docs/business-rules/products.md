# Бізнес-правила: Товари (тури / готелі / сертифікати)

Увесь контент-каталог = **товари WooCommerce**. Новини — дописи `post`, відгуки — CPT `review`.

## Типи через категорії `product_cat`

```
tours (Тури)                ← каталог /tours/ показує всю гілку
├── sea (Тури на море)      ← головна, секція «Тури на море»
├── locations (Локації)     ← головна, секція «Популярні локації»
├── events (Події)          ← головна, секція «Події та свята»
├── weekend, avia, excursion ← «Тип туру» у фільтрі каталогу
hotels (Готелі)             ← /hotels/ і вкладка «Готелі» на сторінці туру
certificates (Сертифікати)  ← банер сертифіката на головній
```

- «Найближчі виїзди» на головній — це **featured**-товари (прапор WooCommerce), а не окрема категорія.
- Каталог виключає `hotels` + `certificates` (`exclude_cats`) доки немає `?cat`.
- Запити централізовані в `mustsee_products()` `inc/products.php:34`.

## Поля туру (мета `_mustsee_*`)

Зберігаються метабоксом «Деталі туру / готелю» — `inc/tour-meta.php:9` (`mustsee_tour_fields`), збереження на `save_post_product` (`inc/tour-meta.php:87`).

| Ключ (`_mustsee_`) | Призначення | Формат |
|---|---|---|
| `days` | Тривалість | «5 днів» |
| `date_range` | Діапазон дат | «14.04 – 19.04» |
| `departure` | Місто виїзду | вільний текст |
| `places` | Міста маршруту | через кому |
| `countries` | Країни | через кому (фільтр «Країна» = LIKE по цьому полю) |
| `city`, `stars` | Готель: місто + зірки | текст / «4» |
| `route` | Маршрут | міста через кому/новий рядок → `mustsee_parse_route()` |
| `departures` | Дати виїзду | по рядку `дата|ціна` → `mustsee_parse_departures()` |
| `program` | Програма | по рядку `День :: крок1 || крок2` → `mustsee_parse_program()` |

Базові поля Woo: назва, **опис** (вкладка «Опис»), **зображення** (головне + галерея), **ціна** (UAH ₴).

## Рендер

- Картка-тур: `mustsee_tour_card()` `inc/products.php:92` (badge днів + ціна, країни, місто).
- Медіа: `mustsee_media()` `inc/products.php:13` — зображення товару або градієнтний `.media-placeholder`, якщо фото немає.
- Ціна: `mustsee_price()` (порожньо, якщо немає WooCommerce).

## Пастки

- На головній бейдж-«країна» в картці = `countries` (fallback `places`), «місто» = `departure` (fallback `city`) — **одне й те саме поле використовується по-різному** в різних секціях, тож результат може відрізнятися.
- `events` показують підзаголовком `places` (там лежить «15 турів») або excerpt.
- Якщо у товара немає фото — скрізь буде градієнт-плейсхолдер (не битий `<img>`).
- Валюта/формат ціни — глобальні налаштування WooCommerce (UAH, ₴ справа, 0 знаків).

## Демо-дані

`inc/seed-tours.php` (товари + категорії, ідемпотентно по slug) та `inc/seed-images.php` (WebP-плейсхолдери). Запуск:
`wp eval-file wordpress/wp-content/themes/mustsee/inc/seed-*.php`.
